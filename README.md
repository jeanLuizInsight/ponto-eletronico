# Ponto Inteligente
Projeto utilizado no curso de API RESTFul com Spring Boot e Java 8, onde é realizado a criação de uma API a ser utilizada em um sistema de marcação de ponto inteligente.

Tecnologias utilizadas:
* Java 8;
* Stack de Spring;
* Flyway DB;
* ...
